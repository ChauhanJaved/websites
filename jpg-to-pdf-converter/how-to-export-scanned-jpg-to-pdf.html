How do I change a scanned JPGs to a PDF file?

Over the years, Adobe's PDF file type has become a universally accepted method for sharing digital documents. The format's cross-platform adoption means the documents can be viewed on almost any mobile device or computer.

Using our converter, it's possible to create a single multi-page PDF document out of several separate JPG image files. The feature is particularly useful if you need to share a number of scanned documents over email or digitize something for reference.

Download JPG to PDF Converter and install it on your computer. A full version of the program is available free for 15 days. Run the application, you can see the above screen.

Click on “Add Files” or “Add Folders” to add JPG files to the conversion list. You can also drag and drop files and folders from Windows Explorer.

After adding JPG files for conversion, you can directly click on "Convert" button to start conversion. Before that you can change the Options like output folder, output mode and jpeg quality. You can change JPG image order by clicking "Move Up" or "Move Down" and rearrange images in output PDF file. You can merge as many JPG files as you need into a single PDF file. For that you have to select output mode as “Combine all JPG to one PDF”. If you want to create single page PDF file for each JPG image then you have to select “Create separate PDF for each JPG” option.

Now to transform all JPEG to PDF, Click “Convert” button to begin the batch conversion process. All your output PDF documents will appear in the output folder you have selected.
