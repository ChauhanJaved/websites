License Information

The licensing model is based on the way you are going to use our component in production. For development purpose you can use the downloaded demo version which has all the features enabled for an unlimited period of time. We encourage you to test the component both in development and production environments and we offer free technical support by email to get our component working in your environment.

Our licensing model consists in two types of licenses and you'll have to decide which one best fits your needs.

Deployment License:

A deployment license allows you to use our component in a single web or windows application for personal use or you can deploy/distribute only inside your company.

Redistributable License:

A redistributable license allows you to use our component in a single web or windows application that you can deploy/distribute inside or outside your company, on an unlimited number of servers or to an unlimited number of customers.

You are required to ensure that the component is not distributed in any form that allows it to be reused by any application other than that with which you distribute it. For example, if you install 'Websites Screenshot' component along with a packaged application on a customer's server, that customer is not permitted to use the component independent of your application.
