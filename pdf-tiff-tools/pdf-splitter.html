PDF Splitter

PDF Splitter is a very simple and fast windows utility program that lets you split multi-page PDF files into smaller PDF files. PDF Splitter provides options for how your large PDF documents are split ï¿½ by pages, by page range etc. So if you have a multi page PDF document and you want to reorganize the pages, this tool will do the task. For instance it can split 100-page PDF file and create a new file by every 20 pages. So you get 5 PDF documents with 20-pages each. PDF Splitter does NOT require Adobe Acrobat.
