Windows Command Line Tool to convert html and mhtml files to images in batch, take full length screenshots of multiple websites, generate thumbnail image preview of web pages from any url. Supports JPG, GIF, PNG, BMP and TIF image formats.

Command Line Usage

Key Features

  • Generate thumbnail previews of web pages.
  • Create full length screenshots of website pages.
  • Convert MHTML and HTML pages to images in batch.
  • Take website screenshots automatically on scheduled time.
  • Supports JPG, GIF, PNG, BMP and TIF image formats.
  • Supports JPEG quality (compression) percent.
  • Supports capturing sites with flash content.
  • Supports timeout seconds of web page to load.
  • Supports delayed snapshots of web page.
  • Works in background, web pages don't need to be displayed or kept active to capture.
  • Works in multi threaded batch mode, which enables to capture multiple websites at a time.
  • Switches to disable java, activex, scripts on the web page.

Overview

Convert HTML to Image is a Windows Command Line Tool (Console Application) that helps you to generate thumbnail images of website pages, take full size screenshots of websites, convert mhtml and html pages to image. It captures the whole web pageï¿½s content into JPEG, PNG, Bitmap, GIF or Tiff image on the fly.

Command Line Usages:

Create full size screenshot from single website url (online page).
>ConvertHTMLtoImage.exe /url "www.ConvertHTMLtoImage.com"

Generate thumbnail image of web page.
>ConvertHTMLtoImage.exe /url "www.ConvertHTMLtoImage.com" /width 100 /height 200

Convert HTML to image (offline page).
>ConvertHTMLtoImage.exe /file "c:/htmlfile.html"

Capture multiple webpages in batch (online or offline pages).
>ConvertHTMLtoImage.exe /list "c:/list.txt"
