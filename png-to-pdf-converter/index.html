PNG (Portable Network Graphics) is a bitmapped image format and video codec that employs lossless data compression. It was created originally to replace the GIF file format when it was announced that the companies who created the GIF format wanted royalties. PNG uses indexed colours and lossless compression (like a .GIF file), but without those copyright limitations; it cannot be animated like a GIF image. As PDF (Portable Document Format) files are compact and highly portable, many users prefer to store PNG in PDF format. PDF is one of the most versatile formats, because it combines graphic, images and text in a resolution independent and device independent extension. Some users prefer to convert PNG images to PDF in order to save disk space and ease file transmission over the internet.
